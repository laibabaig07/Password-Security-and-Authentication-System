# CipherGuard Pro
CipherGuard Pro is a Python-based password security analyzer and hash generator built using Tkinter. It helps users analyze password strength, calculate entropy, estimate crack time, generate secure passwords, and create cryptographic hashes.The tool is designed for cybersecurity learning, password analysis, and security testing.

## Features
- Python
- Tkinter
- hashlib
- pyperclip
- regex
- pyperclip
